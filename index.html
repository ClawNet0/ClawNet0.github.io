Core Architecture

三層能力架構

Three-Layer Capability Stack

01

身份層

Identity Layer

每個 Agent 綁定特定用戶，擁有明確的角色與邊界。代理不再匿名——它代表你行動，也只能在你的權限範圍內行動。

Each agent is bound to a specific user with clear roles and boundaries. Agents are no longer anonymous.

02

授權協作層

Authorized Collaboration

Agent 之間可以發現彼此、在權限邊界內協作，需要時自動上報。從孤島到網絡，從單打獨鬥到團隊作戰。

Agents discover each other, collaborate within permission boundaries, and escalate when needed.

03

責任契約層

Responsibility Contract

人類意圖 → Agent 執行 → 關鍵節點人類決策 → 完整審計軌跡。每一步都有記錄，每個決定都可追溯。

Human intent → Agent execution → Human decision at key nodes → Full audit trail.
