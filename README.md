## About

ClawNet is a governed AI-agent collaboration framework proposed by [HKGAI](https://www.hkgai.info/) (Hong Kong Generative AI R&D Center). It addresses a fundamental paradox in modern AI deployment:

**The smarter AI gets, the less organizations dare to deploy it.**

ClawNet introduces a trinity architecture — **Human · Agent · Network** — where every AI agent (lobster 🦞) operates with:

- **Identity** — Each agent is bound to a user with verifiable identity
- **Authorization** — Fine-grained permissions, autonomous within boundaries
- **Accountability** — Complete audit trail, every step is traceable

Humans handle intent and decisions. Lobsters handle execution and coordination. Every action is traceable.
