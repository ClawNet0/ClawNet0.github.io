GOG Agent Skill 隱私權政策

最後更新日期：2026年3月31日

ClawNet 團隊為您提供 GOG Agent Skill（以下簡稱「本 Skill」），這是一套專為 AI Agent 設計的 Google Workspace 整合工具。我們深知隱私的重要性，並制定了最嚴格的數據處理原則。

核心聲明：本 Skill 為本地端/客戶端執行的工具。ClawNet 開發團隊「絕對不會」收集、攔截、傳輸或儲存您的任何 Google 帳戶數據。

1. 數據的獲取與處理方式

當您授權本 Skill 訪問您的 Google 帳戶（如 Gmail、Calendar、Drive 等）時：

  • 直接通訊： 所有的 API 請求均由您運行的 AI Agent 或本地環境直接發送至 Google 伺服器。
  • 本地儲存： 您的 OAuth 憑證 (Tokens) 僅加密儲存於您所使用的 Agent 運行環境或本地設備的 Keyring 中，絕不會上傳至 ClawNet 的伺服器。

2. 我們收集的資訊

我們不收集任何資訊。 由於系統架構的設計，ClawNet 無法存取您的電子郵件內容、日曆事件、雲端硬碟檔案或任何個人身份資訊 (PII)。

3. 數據分享與第三方披露

因為我們未持有您的任何數據，所以不存在向廣告商、第三方機構分享或出售您數據的可能性。

4. 如何撤銷授權

您可以隨時前往您的 Google 帳戶安全性設定中的「第三方擁有帳戶存取權的應用程式」，手動移除 GOG Agent Skill 的授權。移除後，您的 Agent 將立即失去訪問您 Google 數據的能力。
